# Guide d'Installation PWA sur iPhone

## Installation de MeteoSensei comme application iPhone

### Étape 1 : Ouvrir dans Safari
1. Ouvrez Safari sur votre iPhone
2. Accédez à l'URL déployée sur GitHub Pages
3. Attendez que la page se charge complètement

### Étape 2 : Ajouter à l'écran d'accueil
1. Tapez sur le bouton "Partager" (icône avec flèche vers le haut) en bas de l'écran
2. Faites défiler et sélectionnez **"Sur l'écran d'accueil"**
3. Personnalisez le nom si désiré (ex: "Météo QC")
4. Tapez sur **"Ajouter"** en haut à droite

### Étape 3 : Activer les notifications
1. Ouvrez l'application depuis l'écran d'accueil
2. Dans la section "Notifications", tapez sur **"Activer les notifications"**
3. Autorisez les notifications quand demandé par le système

### Fonctionnalités activées
✅ **Mode standalone** : L'app s'ouvre sans la barre d'adresse Safari  
✅ **Icône personnalisée** : Gradient violet avec icône soleil  
✅ **Notifications push** : Alertes si pluie prévue  
✅ **Cache offline** : Fonctionne sans connexion après premier chargement  
✅ **Thème iOS** : Barre de statut adaptée aux couleurs de l'app

### Notes importantes

#### Pour les notifications sur iPhone :
- Les notifications fonctionnent uniquement si l'app est ajoutée à l'écran d'accueil
- Safari classique ne supporte pas les notifications push
- iOS affiche les notifications même si l'app est fermée

#### Mise à jour de l'app :
- Les mises à jour sont automatiques au lancement
- Pour forcer une mise à jour : fermer l'app complètement et la rouvrir

#### Suppression :
- Maintenir l'icône enfoncée sur l'écran d'accueil
- Sélectionner "Supprimer l'app"

---

## Test des notifications

1. Sélectionnez une ville
2. Activez les notifications
3. L'app vérifie automatiquement s'il va pleuvoir aujourd'hui
4. Si oui, une notification est envoyée (max 1/jour)

Pour tester manuellement :
- Changez de ville pour déclencher une nouvelle vérification
- Les notifications s'affichent même si l'app est en arrière-plan
